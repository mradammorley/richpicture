// CREATING HOTSPOT SVGS
1) Place richpicture image on an illustrator artboard with the same dimensions
2) Draw hotspot using pen tool with any colour fill
3) Edit layer names to be hs1, hs2, etc
3) Export as SVG, choose "Use Artboards", Object IDs - Layer Names
4) Open the SVG using a code or text editor
5) Copy the shape into hotspots.js as a value into the key "svg"
